
## 全文

```
gcc main_base.c
./a.out
```

## ソースを分割した

ヘッダに書いた#ifnfdefはマクロ。コンパイラに指定するもので、２重インクルードを防いでる。    
ヘッダーは2つのファイルから呼ばれたりするが、1回の呼び出しでいいので書いとく。    


```
gcc main.c calc.c
./a.out
```


## ソースを増やした

calc.hヘッダーはあちこちから呼ばれるようにした。全部のc langファイルをgccで指定しないとエラーになる。


```
gcc main2.c calc2A.c calc2B.c calc.c
```
